# Тестовое от АльфаБанка в магистратуру
## Кредитный скоринг
Разработайте модель прогнозирования выхода клиента в дефолт на основании истории карточных транзакций. 
Модель должна качественно отделять потенциальных клиентов-неплательщиков от тех, кто способен своевременно выплачивать полученный займ.
